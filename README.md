# Number Guessing Game

A simple command-line game where the computer picks a random number between 1 and 100, and you try to guess it within a limited number of attempts.

## Features

- Random number generation
- Input validation (handles non-numeric input gracefully)
- Limited attempts with a "too low" / "too high" hint system
- Reveals the answer if you run out of attempts

## What I learned

- Using loops and conditionals for game logic
- Handling user input errors with try/except
- Working with the `random` module
- Setting up Git and pushing a project to GitHub
